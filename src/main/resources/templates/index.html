API para consultas de cep

Uma API para consultar CEP de maneira fácil, rápida e gratuita.

Exemplo consulta:

http://cepapi.delivoro.com.br/14801210

FAÇA SUA consulta:

Digite um CEP para testar a resposta da API.

Exemplo resposta:

{
  "cidade":"Araraquara",
  "logradouro":"Avenida São Geraldo",
  "bairro":"Centro",
  "bairroNormalizado":"centro",
  "cep":"14801-210",
  "tipoLogradouro":"Avenida",
  "tipoLogradouroNormalizado":"avenida",
  "cidadeNormalizada":"araraquara",
  "logradouroNormalizado":"avenida sao geraldo",
  "uf":"SP"
}

Como utilizar?


Exemplo Curl

curl -X GET "http://cepapi.delivoro.com.br/14801210"
curl -s -X GET "http://cepapi.delivoro.com.br/14801210" | jq "."

Exemplo PHP

$response = file_get_contents("http://cepapi.delivoro.com.br/14801210");
print($response);

Exemplo Ruby

require "net/http"

result = Net::HTTP.get URI "http://cepapi.delivoro.com.br/14801210"
puts result

Exemplo JQuery

$.ajax({
  type: "GET",
  url: "http://cepapi.delivoro.com.br/14801210",
  dataType: "JSON",
  success: function(data) {
    alert(data):
  }
});

Exemplo JAVA

import org.apache.commons.httpclient.*;

try {
  GetMethod httpMethod = new GetMethod("http://cepapi.delivoro.com.br/14801210");
  httpMethod.setRequestHeader("Content-type", "application/x-www-form-urlencoded;json;charset=UTF-8");

  HttpClient httpClient = new HttpClient();
  httpClient.executeMethod(httpMethod);

  System.out.println(new String(httpMethod.getResponseBody()));
} catch (Exception e) {
  System.err.println(e.getMessage());
}
